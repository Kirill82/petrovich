About

Some useful information.

Genre

Petrovich is a browser game in which the player character is continuously moving forward through the game world.

Technologies

This game is built using: JavaScript, Canvas, Web Audio API

Resolution

Resolution of the game screen is 1280 x 700, so if your device does not comply with this resolution, you have 3 choices:

1 - close this site and forget it

2 - buy (or steal) a device of the appropriate size

3 - reduce the browser window (ctrl -)

Difficulty

Petrovich is the HARDCORE game for the strong, true gamers. It expects you to fail time and time again.

Controls

Everything as simple as a hammer.

  • JUMP

    Up arrow

    Just press the up arrow on your keyboard to make Petrovich soar up like a bird

  • SLIDE

    Down arrow

    Just press the down arrow on your keyboard to make Petrovich slide like a ninja
